Myo is a Symfony 7.4 LTS experimentation project — used to try out various Symfony features and packages.

**Why:** Personal sandbox for learning and testing, not a production app.

**How to apply:** Favor quick setup and working examples over production-grade architecture. Experimentation-friendly decisions are preferred.
